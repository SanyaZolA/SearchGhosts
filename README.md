## Приложение по поиску духов (SearchGhosts)

Что умеет: 
  Приложение состоит из одной страницы /monitoring.
  Выводит список духов (набор карточек).
  Карточки имеют поля:
    -Имя
    -Изображение
    -Статус
    -Локация
    -Уровень угрозы
  Данные мокаются через Next.js Route Handlers.

  В карточке духа имеется кнопка "Поймать".
  При клике:
    — Отправляется мутация на API.
    — Применяется Optimistic Update (интерфейс обновляется мгновенно).
    — С вероятностью 30% API возвращает ошибку и показывает уведомление.

  Реализована Server-Sent Events (SSE) (Раз в 5 секунд случайный дух должен менять уровень угрозы (например, с "Low" на "Critical")).
  В корне имеется docker-compose.yml, позволяющий запустить проект одной командой: docker-compose up.
