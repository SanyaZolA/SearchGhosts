# SearchGhosts
## Приложение по поиску духов

### Описание / возможности:
- Приложение состоит из одной страницы /monitoring.
- Выводит список духов (набор карточек).
- Карточки имеют поля:
    - Имя.
    - Изображение.
    - Статус.
    - Локация.
    - Уровень угрозы.
- Данные мокаются через Next.js Route Handlers.
- В карточке духа имеется кнопка "Поймать".
- При клике:
    - Отправляется мутация на API.
    - Применяется Optimistic Update (интерфейс обновляется мгновенно).
    - С вероятностью 30% API возвращает ошибку и показывает уведомление.
- Раз в 5 секунд случайный дух меняет уровень угрозы (например, с "Низкого" на "Критический").

  В корне имеется docker-compose.yml, позволяющий запустить проект одной командой: docker-compose up.

### Технологии и стек
- **Core**: React 18+, Next.js (App Router)..
- **Architecture**: Строгий Feature Sliced Design (FSD).
- **State & Async**: TanStack Query.
- **Styling**: SCSS Modules.
- **Validation**: Zod (валидация всех входящих данных).
- **DevOps**. Docker Compose для запуска.
